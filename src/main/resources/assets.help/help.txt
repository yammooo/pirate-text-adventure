Usage: command [options] [arguments]

Description:
  Brief description of what the command does.

Options:
  -h, --help             Display this help message and exit.
  -v, --version          Display version information.
  -f, --force            Force the operation.
  -o, --output <file>    Specify an output file.
  -r, --recursive        Process files recursively.
  -a, --all              Include hidden files.
  -l, --list             List items instead of performing an action.
  -d, --directory <dir>  Specify a directory to work in.
  -p, --path <path>      Specify a path for the operation.

Arguments:
  <file>                 Specify a file to operate on.
  <directory>            Specify a directory to operate on.
  <pattern>              Specify a pattern to match.

Examples:
  command -f file.txt                         # Perform operation forcefully on file.txt
  command -r -o output.txt /path/to/directory # Process files recursively and output results to output.txt in /path/to/directory
  command -l -a /home/user                    # List all files, including hidden files, in /home/user
